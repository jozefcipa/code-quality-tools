# @strv/textlint-config

> STRV's config for [textlint][textlint-home]

## STRV Config

.textlintrc or .textlintrc.json

```json
{
  "rules": {
    "@strv/textlint-rule-preset-strv": true
  }
}
```

.textlintrc.js

```js
module.exports = {
  rules: {
    '@strv/textlint-rule-preset-strv': true
  }
}
```
